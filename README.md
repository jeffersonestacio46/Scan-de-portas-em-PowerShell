# Scan-de-portas-em-PowerShell
Esse Scan de portas abertas em PowerShell, fazendo um Scan completo pelo o IP do alvo informado

Abra seu PowerShell como Administrador e execute o comando a baixo para a shell funcionar

```
Set-ExecutionPolicy Unrestricted
```

Depois de excutar o comando a cima, vá até o diretorio onde está salvo o script 

Ex: > `"C:\Users\Admin\Download"`

No diretorio onde o arquivo foi baixado execute o comando a baixo:
```
.\ScanMetasploittable2.ps1
```
Depois de excutar irá solitar o IP da maquina alvo, coloque o IP para que possa realizar o scan de portas abertas
